1. 일단 해당 브랜치에서는 frontend 폴더에서 리액트 문법을 기반으로 로그인 기능들을 먼저 구현해보려는 것이 목적이야.
일단 notion의 mcp to design 페이지내의 thanksquirrel 페이지에 접근해서
로그인 관련 UI 및 기능들을 구현해주었으면 해
일단 기능 부분은 일반 로그인 먼저 구현해주고, 추후에 간편 로그인(카카오, 네이버, 구글 로그인)도 구현해볼거야
UI에서 일단 다람쥐 캐릭터, "달램이"가 처음에는 로그인 페이지에서 반겨주는 애니메이션을 넣었으면해
"달램이" 이미지는 supabase의 storage에 들어있고, thanksquirrel-image에 모든 사진들이 들어있어
내가 로그인 창에서 입력을 진행하게 되면 "달램이"가 눈을 가리는 사진으로 변경하였으면 해
달램이 이미지는 notion에서도 확장자명을 지정해놓았는데,
일단 로그인 창에서 숨어있을 때에는 기본 이미지(character-web.webp)형태로 로그인 창에서 숨어있다가
다시 반겨주는 이미지(character-hello.webp)로 변경되면서 로그인 창에서 나타나고
로그인 창에서 입력을 진행하게 되면 눈을 가리는 이미지(character-hide.webp)로 변경되도록 하면 좋겠어
1-1. 전체적으로 괜찮기는 한데 일단 UI랑 일부 애니메이션 부분 좀만 바꿔줘, 내가 mcp to design에서
만들어놓은 로그인 UI를 기반으로 만들어보았으면 해 배경화면은 background.webp를 통해 전체적인 배경을 설정해주고
달램이가 눈을 감는 시점은 입력 창에 이메일이나 비밀번호가 입력이 된 상태에서 계속 진행되어야 돼
(만약에 입력창에 이메일이나 비밀번호가 입력이 안된 상태라면 달램이는 눈을 뜨고 반겨주는 애니메이션을 유지해야해)
비밀번호 찾기나 회원가입 버튼도 일단은 만들어만 놔 나중에 내가 거기서 기능을 추가해볼 것이니까
1-2. 그리고 달램이가 말하는 말풍선 같은 것도 추가해줘 참고로 notion의 mcp to design의 thanksquirrel 페이지에서 나오는 말풍선 레이아웃들을 참고로해주면 돼 등장할 때 말풍선도 opacity값을 0에서 1로 주면서 transition도 0.3초 정도로 스르륵 나타나는 식으로 표현하면 좋겠어, 그리고 인사할 때 대사는 "다람다람! 누구신지 알려주시면 감사하겠습니다람!”이라고 표현해주고, 
아이디/비밀번호 입력할 때의 대사는 "다람다람! 안보고있을거니 걱정안하셔도 됩니다람!”으로 표현하면 좋겠어
참고로 말풍선은 자주 쓸 것이니까 컴포넌트 형태로 대사같은 것들 바꿔서도 자주 사용할 수 있도록 만들면 좋겠어
그리고 달램이가 눈을 감는 조건에 이메일/비밀번호 입력창에 값이 입력이 되어있을 때에 더해 각 입력창에 포커스 되어있을 때도 포함해줘
2. 이번에는 회원가입 관련 ui를 만들어보려고 하거든 로그인 창에서 회원가입 버튼을 누르면 회원가입 관련 팝업이 나타나게 해줘 참고로 notion의 mcp to design 페이지의 thanksquirrel에 보면 회원가입 디자인이 있어 그것을 참고해주면 좋아 일단 회원가입창에 필수로 넣어볼만한 것은 이메일(필수 입력값), 비밀번호(필수 입력값, 8자리 이상, 특수문자, 숫자, 영어 포함), 비밀번호 확인(필수 입력값), 별명(이메일 확인 후 해당 이메일이 기본값, 따로 바꾸기 가능), 성별(기본은 비밀, 남자나 여자로 바꿀 수 있음), MBTI(기본값은 infp(supabase 데이터베이스에 기본값으로 존재), 추후에 select로 바꿀 수 있음), 취미(기본값 빈 문자열), 특기(기본값 빈 문자열)가 있어 이 데이터들은 supabase의 데이터베이스에 있어 회원가입에 성공하면 거기서 비밀번호를 제외하고 형식에 맞는 다른 정보들이 다 저장되어야 해 로그인/회원가입 방식은 supabase이 인증 서비스를 통해 할거야 회원가입에 성공해서 데이터가 저장되면 로그인하게 되면 회원가입을 하였을 때의 정보를 기반으로 이메일과 비밀번호를 입력하면 로그인이 되도록 해줘
2-1. 회원가입을 통해 Authentication에서 회원가입했을 때 데이터가 정상적으로는 올라와 그런데 데이터베이스에서는 스키마에 유저 관련 데이터들이 올라오지 않았는데 그 부분에 관해서 회원가입 했을 때 정상적으로 데이터들을 올려놓을 수 있도록 수정해줘, 그리고 회원가입 한 후에 로그인을 하려는데 제대로 된 이메일과 비밀번호를 입력했는데도 로그인이 안되는 현상이 발생하고 있어 그 부분에 관해서도 수정해줘, 그리고 달램이가 로그인 창에서 회원가입 창이 뜨면 눈을 가리도록 해줘, 그리고 회원가입이나 로그인이 완료되면 중앙에 말풍선같은 확인 창이 뜨고, 달램이가 로그인창 말고 밑에서부터 애니메이션으로 올라와서 상체 부위가 보이기까지 만들어줘 참고로 디자인은 notion의 mcp to design 페이지의 thanksquirrel에 보면 관련 디자인들이 있어
